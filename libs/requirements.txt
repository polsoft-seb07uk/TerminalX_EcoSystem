# ── [B] WYMAGANE ZALEŻNOŚCI PIP ──────────────────────────────────────────────

#  Poniższe pakiety są używane przez moduły core i są wymagane

# --- Procesy i system ---------------------------------------------------------

# psutil — wymagany przez core/runner.py (kill drzewa procesów, suspend/resume,
#           priorytety, limity RAM/CPU, jobs top) oraz core/debugger.py
#           (statystyki pamięci procesu). Moduły mają fallback bez psutil,
#           ale funkcje run --top, run --kill, run --priority nie działają.
psutil>=5.9.0

# --- Terminal i UX ------------------------------------------------------------

# colorama — inicjalizacja ANSI na starszych terminalach Windows (ConHost <Win10).
#            core/ansi.py i core/colors.py używają ctypes natywnie;
#            colorama jako fallback dla środowisk bez VT processing.
colorama>=0.4.6
